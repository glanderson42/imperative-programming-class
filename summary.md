# Valtozok

```
int num = 10; // egesz szamok tarolasa 16 bit-en
long int bignum = 10; // egesz szamok tarolasa 32 bit-en
long long int verybig = 10; // egesz szamok tarolasa 64 bit-en

short int small = 10; // egesz szamok tarolasa 8 bit-en

float flt_num = 1.0f; // tort szamok tarolasa
double dbl_num = 1.0; // tort szamok tarolasa (pontosabb, osszehasonlitasra alkalmas)

char c = 'a'; // karakterek tarolasa
char* str = "str"; // 'string' tarolasa
```
